# Pandit Ji Paneer Wale Website

This is a modern, responsive e-commerce website for Pandit Ji Paneer Wale, built with React, Vite, and Tailwind CSS.

## Features

- **Product Listing**: Browse products by category (Spices, Milk Products, etc.).
- **Search & Filter**: Easily find products using the search bar or category filters.
- **Shopping Cart**: Add items to cart and manage quantities.
- **WhatsApp Checkout**: Send your order directly to WhatsApp for quick processing.
- **Responsive Design**: Works perfectly on mobile, tablet, and desktop.
- **Modern UI**: Beautiful animations and a clean, colorful interface.

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```
    Open your browser and navigate to the URL shown (usually `http://localhost:5173`).

3.  **Build for Production**:
    ```bash
    npm run build
    ```
    The output will be in the `dist` folder.

## Customization

- **Products**: Edit `src/data/products.js` to add or modify products.
- **Colors**: Edit `tailwind.config.js` to change the brand colors.
- **Contact Info**: Update `src/components/Footer.jsx` and `src/components/Cart.jsx` with your contact details.

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion (Animations)
- Lucide React (Icons)
